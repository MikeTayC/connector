# Welcome

Welcome to the Service Cloud Connector wiki! The wiki itself is actually a git repository, which means you can clone it, edit it locally/offline, add images or any other file type, and push it back to us. It will be live immediately.

Go ahead and try:

```
$ git clone https://bitbucket.org/demandware/service-cloud-connector.git/wiki
```

# Overview

   'Service cloud connector' or SCC is a community accelerator package to integrate commerce cloud and service cloud. SCC has following two components: 
   
1. Service Cloud Package : You can install pre-built package. 
2. Commerce Cloud Cartridge : Cartridge needs to be deployed on commerce cloud instance.
